# Livraria

## Tecnologias

* Arquitetura MVC;
* Design Pattern Front Controller;
* JDBC;
* Design Pattern DAO; 
* POJOs; 
* Banco de dados Oracle 11G;
* JSTL;
* HTML5;
* CSS3;
* Servidor Tomcat;
* IDE Eclipse Mars.
