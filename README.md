# Livraria

## Tecnologias

* Java 7; 
* Arquitetura MVC;
* Design Pattern Front Controller;
* Design Pattern DAO;
* Design Pattern Factory; 
* Servlet;
* JSP;
* JDBC;
* POJOs;
* JSTL;
* HTML5;
* CSS3;
* Banco de dados Oracle 11G;
* Servidor Tomcat;
* IDE Eclipse Mars;
* UML.
